To build/import the project into Eclipse:

1)Ensure the latest java is installed on the machine and extract the zip folder
2)Create a new project by going to File -> New -> Java Project
3)Enter a project name and click finish
4)If prompted to create new module-info.java, select Don't Create
5)Right click on the created project, select Import -> General -> File System
6)Click Browse next to the From directory, open Island Trader -> code and tests -> island_trader
7)Tick the box next to the island_trader folder and click Finish
8)If prompted to overwrite anything, click Yes To All


To run the JAR file:

1)Ensure the latest java is installed on the machine and extract the zip folder
2)Open the exported folder in a terminal window
3)Run "java -jar Islandtrader.jar"
